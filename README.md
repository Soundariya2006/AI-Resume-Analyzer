#  AI Resume Analyzer

A web-based Resume Analyzer developed using **Python** and **Streamlit** that analyzes PDF resumes, extracts technical skills, calculates a rule-based ATS score, and provides improvement suggestions.

---

##  Features

-  Upload PDF Resume
-  Extract Resume Text
-  Detect Technical Skills
-  Calculate Rule-based ATS Score
-  Resume Improvement Suggestions
-  User-friendly Streamlit Interface

---

##  Tech Stack

- Python
- Streamlit
- PyMuPDF
- Pandas
- Git
- GitHub

##  Installation

1. Clone the repository

```bash
git clone https://github.com/Soundariya2006/AI-Resume-Analyzer.git
```

2. Navigate to the project

```bash
cd AI-Resume-Analyzer
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
streamlit run app.py
```
